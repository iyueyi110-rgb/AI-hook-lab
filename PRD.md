# AI Hook Lab PRD

## 定位与阶段

这是验证“结构化输入—多版生成—模型自评分—人工收藏/采用—Bad Case 复盘—Prompt 迭代”的个人 AIGC 工具 Demo，不宣称模型优于通用大模型，也不把模型评分当成真实点击率。

## 核心角色与主链

- 创作者：输入主题、平台、内容类型、目标用户、情绪和字数，收藏、复制、采用并提交人工满意度。
- 产品运营者：查看来源隔离的事件、Bad Case 和 Prompt 版本表现，决定是否采纳 candidate。
- 模型：生成候选 Hook、理由与自评分，不替代人工发布决策。

主链：结构化输入 → 输入校验与隐私提示 → baseline/candidate 生成 → Schema 与内容校验 → 模型自评分排序 → 人工收藏/采用/满意度 → 来源隔离看板 → Bad Case 归因 → 下一版 Prompt。

## 验收

生成失败不覆盖历史；事件可持久化；评测与真实操作不混算；模型分与人工分分开；同一评测样本可成对比较；没有 API Key 时明确失败，不能伪造生成结果。
