## 投递前检查

1. 使用手机流量、无痕浏览器和桌面浏览器分别访问。
2. 确认 30 秒内能理解产品并进入经典生成主路径。
3. 检查无 Key、上游失败、请求超限和移动端错误态。
4. 录制 60–90 秒备用演示：问题与用户 → 输入 → 10 个候选 → 选择/反馈 → 评测边界。
5. 地址不稳定时，简历优先使用 GitHub、截图和录屏，不放失效链接。
